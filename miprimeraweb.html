Esta es mi primera página web.

Es una web muy simple, sin grandes pretensiones. Solo queremos ver cómo funciona el HTML.

¿Qué me permite hacer el html? Puedo insertar un link a otra web, escribir cosas en cursiva, en negrita o incluso en cursiva y en negrita a la vez.

También puedo insertar una imagen que me guste:


O crear una tabla con datos:

100 200 300
400 500 600

Fijaos en la barra de direcciones del navegador, donde normalmente escribimos las URL de la web a la que queremos navegar. Como veis, el navegador no está leyendo una página en internet, sino una página que está en nuestro ordenador, es decir "en local". El navegador está leyendo el código HTML del archivo, interpretándolo y mostrándonos lo que contiene. Pero ese HTML no está en internet disponible para cualquiera, ese HTML lo está leyendo desde nuestro propio ordenador.
